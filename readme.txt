Welcome to LinkedFootball
==========================

This project provides a web application to access and query football data using Semantic Web and Linked Data technologies.

Follow the instructions below to run the application:

1. Running the Application Directly
-----------------------------------
   You can open the application directly in your browser without running a server:
   1. Navigate to the project folder where you cloned or downloaded it.
   2. Open the `index.html` file in any modern browser.

2. Accessing the Application Online
-----------------------------------
   The application is also hosted online using GitHub Pages. You can access it directly using the following URL:
   https://afonsom1719.github.io/LinkedFootball


Enjoy exploring LinkedFootball!
